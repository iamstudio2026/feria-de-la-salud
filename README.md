# Feria de la Salud - IAMStudio2026

Conjunto de aplicaciones web para la Feria de la Salud CeCoSaBi-DGCOSEM.

## Estructura del Proyecto

- apps/feria-salud/    -> Dashboard principal de la Feria de la Salud
- apps/casaxcasa/      -> App Casa x Casa - Barreras de salud
- apps/gane-gane/      -> Modelo de Cuidado Integrado Gane-Gane
- apps/sarampion/      -> Sistema Comando Sarampion
- assets/images/       -> Logos e imagenes institucionales
- assets/docs/         -> Documentos PDF y DOCX de referencia
- _temp/               -> Scripts y archivos temporales de desarrollo
